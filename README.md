# Decodificador de texto: Alura challenge
Um codificador/decodificador de textos, que substitui as **vogais** por combinações de letras, conforme mostra a tabela abaixo.

| VOGAL | SUBSTITUIDA POR: |
|-------|------------------|
| A     | AI               |
| E     | ENTER            |
| I     | IMES             |
| O     | OBER             |
| U     | UFAT             |
